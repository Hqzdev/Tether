# Tether v1.6.1

Release date: 2026-06-20

## Changes

- Added structured diagnostics across the local proxy, web submission routes, and macOS networking flows without logging provider keys, prompts, responses, or feedback bodies.
- Centralized backend settings validation for themes, proxy ports, profile names, and provider keys.
- Reworked feedback and waitlist route handling with request-level tests for JSON submissions, form submissions, validation failures, and honeypot no-op behavior.
- Updated the public download copy so the site references the current macOS alpha release.

## Notes

This release focuses on safer observability, clearer validation boundaries, and a small release-text correction for the macOS alpha download section.
