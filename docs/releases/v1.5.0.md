# Tether v1.5.0

Release date: 2026-06-18

## Changes

- Redesigned the inspector replay controls with a clearer primary/secondary action hierarchy and native macOS-style buttons.
- Improved the right inspector header with a larger selected-node title, clearer status badge, and more readable model metadata.
- Updated graph cards with more padding, better latency/token layout, hidden zero-cost labels, and animated rounded connection paths.
- Fixed scroll handling so the graph canvas no longer steals scroll input from the inspector and sidebar.
- Reworked the sidebar call rows with a stronger status rail and less visual noise.
- Replaced the landing-page hero video with a larger embedded cinematic product preview.
- Removed the cinematic intro wordmark and marketing phrases so the landing preview opens directly on the app interface.

## Notes

This release focuses on UI clarity across the desktop app and a cleaner landing-page product preview for macOS users evaluating Tether.
