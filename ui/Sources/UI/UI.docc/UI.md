# UI

@Metadata {
    @TitleHeading("Framework")

UI contains reusable SwiftUI design-system primitives and feature components
used by the Tether macOS application.

## Overview

The visual language should stay compact and instrument-like: clear hierarchy,
stateful color, restrained glass surfaces, and controls sized for repeated
debugging work rather than marketing presentation.
